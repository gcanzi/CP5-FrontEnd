# App Melodia — Landing Page

**Check-Point 05 | Front-end Design**

Landing page de página única para o aplicativo de músicas **App Melodia**, desenvolvida como entrega do CP5 da disciplina de Front-end Design.

---

## Tecnologias Utilizadas

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura semântica da página |
| Tailwind CSS (CDN) | Estilização responsiva e utilitária |
| CSS3 (`style.css`) | Ajustes finos, variáveis e efeitos customizados |
| Font Awesome 6 | Ícones em todas as seções |
| Google Fonts (Inter) | Tipografia moderna |
| JavaScript Vanilla | Comportamento do navbar e formulário |

---

## Estrutura de Arquivos

```
CP5-FrontEnd/
```

---

## Seções da Página

1. **Navbar** — Menu fixo com efeito de transparência ao rolar
2. **Hero** — Título, subtítulo, CTA "Ouvir Agora" e mockup do app
3. **Benefícios** — 4 diferenciais com ícones (qualidade, playlists, descoberta, interface)
4. **Funcionalidades** — 6 cards de recursos do app
5. **Depoimentos** — 3 avaliações de usuários fictícios com avatares
6. **Contato** — Formulário de captura de e-mail para marketing
7. **Footer** — Redes sociais, contato e copyright
